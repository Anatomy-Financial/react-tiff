# react-tiff-viewer

## Install

```bash
npm install --save react-tiff-viewer
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'react-tiff-viewer'
import 'react-tiff-viewer/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
